Module 1 · Report Parts

Learn the structure of a formal report

⚡ Up to 60 XP
📚 What is a Report?
📌 Key Idea

A report is a formal document that gives information about a topic. It has a clear structure with different parts. Each part has a special job to do.

⭐ A formal report has 5 main parts
1 · Title
What is the report about?Example: A Report on Social Media Use Among HCT Students
2 · Introduction
Why was the report written? Who? What? How?Background · Purpose · Scope · Method
3 · Findings
What did you find out? (Describe the data)Use charts and statistics to show results
4 · Conclusion
What do the findings mean?Summarise the most important points
5 · Recommendations
What should happen next?Give advice based on the findings

Module 2 · Writing the Title

Master the full formula — Type + Purpose + 5W+1H

⚡ Up to 140 XP
📚 Step 1 — Know Your Report Type
📌 Key Idea

There are 3 types of report. The type must appear in your title. Each type has its own purpose verb — the verb that comes after "to".

Type 1
📋 Informative Report
Collects and presents facts. No problem to solve.
Purpose verbs: to Analyse / to Examine / to Investigate
Use "An" (vowel sound)
Type 2
🔧 Problem-Solving Report
Identifies a problem and suggests solutions.
Purpose verbs: to Address / to Solve / to Resolve
Use "A" (consonant)
Type 3
📊 Evaluation Study Report
Measures and compares how well something is working.
Purpose verbs: to Compare / to Evaluate / to Assess
Use "An" (vowel sound)
📚 Step 2 — The Full Title Formula (5W+1H)
📌 Remember: A title tells the FULL story!

Your title must include: Type + Purpose + What + Who + Where + When + How

✨ COMPLETE TITLE FORMULA
An/A [TYPE] Report to [VERB] the [WHAT] of/by/among [WHO] in [WHERE] in [YEAR] Using [METHOD].
⚠️ Don't forget the full stop (.) at the end!
Colour-coded breakdown:
An Informative Report to Analyse the Study Habits of HCT Students in Abu Dhabi in 2024 Using Survey Methods.
🟢 Report Type 🟡 Purpose (to + Verb) 🔵 WHAT 🟣 WHO 🔴 WHERE 🩵 WHEN 🟠 HOW (method)

Module 3 · Writing the Introduction

Learn to write 4 key sentences — one by one

⚡ Up to 80 XP
📚 The Introduction — 3 Key Sentences
📌 Key Idea

The introduction is one short paragraph with 3 sentences. Each sentence has a job. Memorise this template — you can use it in any report!

Sentence 1
Type + Who Requested It
Tell the reader what type of report it is and who asked for it.
"This is an [Informative / Problem-Solving / Evaluation] report, requested by [name or title]."
Example: "This is an Informative report, requested by the college manager."
Sentence 2
Purpose + Goal
Say WHY the report was written and what it will help to achieve.
"The purpose of this report is to [verb: analyse/address/compare] [topic] to help [goal/reason]."
Example: "The purpose of this report is to analyse the social media use of HCT students to help improve digital wellbeing."
Sentence 3
Overview of the Report
Tell the reader what sections are included in the report.
"It includes an overview of [key sections], with each section providing insights and data that support the main findings and conclusions."
Example: "It includes an overview of the survey methods used, the key findings, and specific recommendations for improvement."
🔑 The Introduction Template — Memorise This!
✨ TEMPLATE
S1  This is an/a [type] report, requested by [name/title].
S2  The purpose of this report is to [verb] [topic] to help [goal].
S3  It includes an overview of [sections], with each section providing insights and data that support the main findings and conclusions.
Full example — Informative Report
This is an Informative report, requested by the training supervisor. The purpose of this report is to analyse the effectiveness of training for factory workers to help improve skill development and productivity. It includes an overview of the survey methods used, findings from worker feedback, and an analysis of key areas of improvement. Each section provides insights and data that support the main findings and conclusions.

Module 4 · Writing Findings

Describe charts and data — sentence by sentence

⚡ Up to 160 XP
📚 How to Write Findings
📌 Key Rule

In the Findings section, you describe what you see in the chart. Use numbers and percentages. Use formal language only. Do NOT give your opinion.

📊 SENTENCE 1 — Opening (who was surveyed)
In our survey, we asked [number] [who] about [topic].
In our survey, we asked 50 HCT students about their social media habits.
📊 SENTENCE 2 — Key Finding (highest result)
According to Figure 1, [%]% of [who] [verb] [what], making it the most common response.
According to Figure 1, 65% of students used Instagram daily, making it the most common response.
📊 SENTENCE 3 — Second Finding (linking word)
Furthermore / Additionally, [%]% of [who] [verb] [what]. (See Figure 1)
Furthermore, 45% of students used Snapchat, while only 12% preferred Facebook. (See Figure 1)
📊 SENTENCE 4 — Lowest / Contrast
In contrast, only [%]% of [who] [verb] [what], making it the least common response.
In contrast, only 12% of students used Facebook, making it the least popular platform.
📌 Useful Language Bank
📈 High Numbers
the majority of (50%+)
most students (60%+)
almost all (90%+)
nearly three-quarters (75%)
📉 Low Numbers
a minority of (<30%)
only a few (<20%)
fewer than half
less than one-third
🔗 Linking Words
Furthermore, ...
Additionally, ...
In contrast, ...
In addition, ...
However, only ...
📋 Reference
(See Figure 1)
(See Chart 1)
As shown in Figure 1,
The data in Figure 1 shows...

Module 5 · Conclusion & Recommendations

Summarise findings and give advice

⚡ Up to 60 XP
📚 Conclusion vs Recommendations
🔚 Conclusion

Summarise the most important findings. What did your data show? Keep it short — 2 to 3 sentences.

Formula
Overall, the findings show that [main result].
Overall, the findings show that the majority of students spend too much time on social media.
💡 Recommendations

Give advice based on the findings. Use modal verbs like should / could / it is recommended that.

Formula
It is recommended that [who] [should + action].
It is recommended that students should limit their daily screen time to 2 hours.
